## Codebase Patterns
- 后端使用 `uv` 管理依赖，开发命令 `cd backend && uv sync --extra dev`；启动 `uv run uvicorn app.main:app --reload --port 8000`
- 后端路径解析基于 `app/core/config.py` 中的 `PROJECT_ROOT`（`Path(__file__).resolve().parents[3]`），所有 workspace 路径常量集中在此
- workspace 目录在运行时通过 `app/core/workspace.py` 的 `ensure_workspace()` 自初始化（lifespan 事件触发），无需手工创建或提交 runtime 文件
- FastAPI 使用 application factory 模式 `create_app()`，便于 TestClient 测试
- CORS 白名单为 `http://localhost:5173`（前端 Vite 默认端口）
- 根 `.gitignore` 忽略整个 `workspace/`，workspace 内的 `.gitignore` 由 `ensure_workspace()` 在运行时生成
